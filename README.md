# Clever Cloud Internship - Desafío Inicial

Este repositorio contiene la solución a los tres ejercicios del desafío técnico utilizando **JavaScript**.

## Ejercicios

1. **Suma de dígitos**
   - Calcula la suma de todos los dígitos de un número entero.

2. **Palíndromos**
   - Verifica si una cadena de texto es un palíndromo.

3. **Ordenamiento**
   - Ordena un arreglo de números enteros en orden ascendente sin utilizar funciones nativas como `sort()`.

## Requisitos

- Node.js instalado.

## Cómo ejecutar

1. Clona o descarga el proyecto.
2. Abre una terminal en la carpeta del proyecto.
3. Ejecuta el siguiente comando:

```bash
node desafio.js
```

## Salida esperada

El programa mostrará en la consola los resultados de los casos de prueba incluidos para cada ejercicio.
