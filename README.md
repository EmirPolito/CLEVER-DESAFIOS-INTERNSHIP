# Clever Cloud Internship - Desafíos 

Este repositorio contiene la solución a los tres ejercicios del desafío técnico utilizando **JavaScript**.

## Ejercicios

1. **Suma de dígitos**: Calcula la suma de todos los dígitos de un número entero.

2. **Palíndromos**: Verifica si una cadena de texto es un palíndromo.

3. **Ordenamiento**: Ordena un arreglo de números enteros en orden ascendente sin utilizar funciones nativas como `sort()`.

## Requisitos

- Node.js instalado.

## Instalación y ejecución

1. Clona este repositorio:

```bash
git clone https://github.com/EmirPolito/CLEVER-DESAFIOS-INTERNSHIP.git
```

2. Entra al directorio del proyecto:

```bash
cd CLEVER-DESAFIOS-INTERNSHIP
```

3. Ejecuta el programa:

```bash
node nombre-archivo.js
```
